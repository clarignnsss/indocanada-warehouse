# IndoCanada Warehouse

## Development Stacks
- [Vue](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [PrimeVue](https://primevue.org/)

## Getting Started
- Clone this repository

- Move to repository
  ```
  cd indocanada-warehouse
  ```

- Install dependencies
  ```
  npm install
  ```

- Run for development
  ```
  npm run dev
  ```
